# Creative_Dev 

Nous avons choisi les thèmes de la tendresse, de la peur et du deuil. 
## Synopsis

Notre projet explore la tendresse, la peur et le deuil à travers deux personnages. Initialement, ils tentent de s'étreindre, mais une force irrésistible les sépare de nouveau. La peur surgit lorsque l'un d'eux est assailli par des mains anonymes. Son compagnon, impuissant, tremble d'angoisse. La scène s'achève sur le deuil : le survivant se retrouve seul dans l'obscurité, avec une simple bougie pour unique lumière.

## Instructions pour démarrer le projet :

- Récupérer le projet : Ouvrez les fichiers dans Visual Studio Code (ou un autre IDE disposant d'une extension équivalente à "Live Server").
- Lancer l'aperçu : Activez l'extension Go Live pour héberger le projet localement.
- Lancer l'expérience : Cliquez sur le bouton pour démarrer l'histoire, puis laissez-vous guider simplement en scrollant (faisant défiler la page).
