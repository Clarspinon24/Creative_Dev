# Creative_Dev 

Nous avons choisi les thèmes de la tendresse, de la peur et du deuil. 
## Synopsis
Nous avons deux personnages qui veulent se prendre dans les bras, mais ils s'éloignent et on ne peut rien faire pour les empêcher de se séparer. Ensuite, l'un des deux se fait attaquer : des mains essayent de l'attraper. On voit que l'autre personnage est inquiet et tremble. Enfin vient le deuil, où l'on voit celui qui s'inquiétait avec une bougie dans le noir.
